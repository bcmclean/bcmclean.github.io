Therapist Jen is a ChatBot that is designed to help users work through negative thoughts and feelings. It is based on Cognitive Behavioural Therapy techniques. Not only does it provide users with an outlet to express their negative emotions, but it also encourages users to create and work towards positive goals. This tool does not replace professional therapy, but is designed to work alongside it.

Tools Used:

  • Python
  • JavaScript
  • SQL

The Maintenance Report App is an Android application that was designed with the University of British Columbia’s Okanagan Campus in mind. The app’s purpose is to create a convenient app where students, faculty, or guests could quickly report maintenance issues around campus (such as lights not working, no toilet paper in the bathroom, etc). The app is meant to handle reports from both guest users and users who want to login with their CWL.

Tools Used:

  • Java
  • Android Studio
